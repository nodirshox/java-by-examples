# Learning Java language by examples

Compile java file in Ubuntu (make sure you have installed JDK).

1.

```
javac HelloWorld.class
```
2.
```
java HelloWorld
```

Attention: class name and file name should be same.

---

1. Hello World

4. Data types

6. Operators
7. Strings
8. Math
9. Boolean
10. Conditions
11. Switch
12. While
13. For Loop
